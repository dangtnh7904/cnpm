# Dự án Quản lý Chung cư (Nhóm 33)

Dự án này được phát triển bằng Spring Boot (Backend) và React (Frontend) với CSDL SQL Server.

## Cấu trúc Dự án

- `/backend`: Mã nguồn Spring Boot API.
- `/frontend`: Mã nguồn React App.
- `/database`: Các file script SQL (tạo bảng, dữ liệu mẫu).
- `/docs`: Tài liệu dự án (ERD, yêu cầu).
